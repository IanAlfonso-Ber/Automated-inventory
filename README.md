🔎 Overview
This script bridges the gap between local stock management and real-time market data. By monitoring your inventory.csv, it automatically triggers web-based price research for low-stock items, allowing for data-driven restocking decisions.

✨ Key Features
Inventory Monitoring: Automatically flags items below user-defined thresholds.

Smart Web Scraping: Navigates search engines to extract real-time pricing data.

Human-like Interaction: Utilizes dynamic waits and randomized inputs to ensure robust operation.

Clean Data: Processes raw search results into structured, analysis-ready CSV files.
